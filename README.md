# IT Ticket Submission Web Application

This project is a web-based IT ticket submission system that allows users to submit support requests through a form and view submitted tickets on a separate ticket listing page.

The application is designed to simulate a basic help desk workflow, focusing on form handling, server-side processing, and dynamic page rendering.

## Features
- Submit IT support tickets through a web form  
- Store ticket data on the server  
- Display submitted tickets on a dedicated ticket page  
- Simple and intuitive user interface  

## Technologies Used
- Python  
- Flask  
- HTML / CSS  
- Jinja2 Templates  

## How It Works
1. A user fills out and submits an IT ticket form.  
2. The server processes and stores the ticket data.  
3. Submitted tickets are displayed on a ticket summary page.  

## Purpose
This project was created to practice building full-stack web applications, handling form submissions, and integrating backend logic with frontend templates using Flask.

## Setup / Run Instructions
To run the application locally, follow these steps:

1. Install Flask:
pip install flask
2. Run the application:
python app.py
3.Open your web browser and go to:
http://localhost:5000
